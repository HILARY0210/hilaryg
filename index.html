    ¿Què es una pàgina web?

     

    Una Página web es un medio digital para promover productos, servicios o información de interés, se consultan a través de una computadora, tableta o Smartphone. Se elaboran en lenguaje de programación HTML y HTML 5 y pueden contener bases de datos y aplicaciones desarrolladas en Java Script, Ayax, PHP, ASP, etc. Una página web es una herramienta valiosa de mercadotecnia para cualquier empresa que requiera de obtener nuevos clientes promoviendo productos o servicios. Se clasifican en :

    PÁGINAS WEB ESTÁTICAS

    Son sitios pequeños que contienen poca información, no cuentan con gestor de contenido y su uso más común es como folletos virtuales de algún servicio o productos.

    PÁGINAS WEB DINÁMICAS

    Son sitios pequeños que cuentan con un gestor de contenido que permite al administrador del sitio modificar la información en cualquier momento sin contar con conocimientos de programación y diseño.
